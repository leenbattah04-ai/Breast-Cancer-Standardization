# Breast Cancer Data Standardization

## 📌Overview
This project demonstrates data preprocessing techniques using the Breast Cancer dataset from Scikit-learn.

The main focus is feature scaling using Standardization (Z-score normalization).

---

## ⚙️ Steps
- Load dataset
- Convert to DataFrame
- Split data into training and testing sets
- Apply StandardScaler
- Transform data consistently

---

## 🧠 Concept
Standardization transforms data so that:
- Mean = 0
- Standard Deviation = 1

This improves machine learning model performance.

---

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## 🚀 Goal
To understand the importance of feature scaling in machine learning pipelines.
